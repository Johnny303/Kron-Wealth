Scroll-Triggered Curved Path: Implementation Approaches

A gold gradient ribbon that draws itself between sections (About → Approach → Services → Contact) as you scroll

A

SVG Path + stroke-dashoffset (Recommended)

How: A single full-page SVG overlay with a cubic Bezier curve. Uses motion/react's useScroll + useTransform to map scroll progress to stroke-dashoffset, creating a draw-on reveal.

Pros

  • Smooth, precise scroll-linked animation (no jank)
  • Already proven in your Approach section's river-path
  • Uses existing motion/react library — no new deps
  • SVG gradient ribbon via linearGradient + wider stroke
  • Fully responsive — path recalculated on resize

Cons

  • Path coordinates need calculation relative to section positions
  • Full-page SVG overlay needs careful z-indexing
B

CSS Scroll-Driven Animations

How: Uses native CSS animation-timeline: scroll() to animate an SVG path's dashoffset without any JavaScript. Pure CSS approach.

Pros

  • Zero JavaScript — purely declarative
  • Native browser performance
  • No library dependency

Cons

  • Limited browser support (no Firefox/Safari stable)
  • Inconsistent with existing motion/react patterns
  • Harder to coordinate with section positions dynamically
  • Less control over easing and intermediate states
C

Canvas 2D Rendering

How: A full-page <canvas> element that redraws a gold curve on each scroll frame using requestAnimationFrame.

Pros

  • Maximum rendering control
  • Can add complex particle effects easily
  • Good for very complex animations

Cons

  • Overkill for a single curved path
  • Not SEO/accessibility friendly
  • Doesn't match existing SVG/motion patterns
  • Higher battery/CPU usage on mobile

My Recommendation: Approach A

SVG + stroke-dashoffset with motion/react is the natural fit. It reuses your existing animation infrastructure, is well-supported across browsers, and creates exactly the draw-on effect you want. The gradient ribbon effect is achieved with SVG's <linearGradient> applied to the stroke, with a wider stroke-width (~10-15px).
