Key Implementation Details

New Component

ScrollPath.jsx — A single full-page SVG positioned with position: fixed, pointer-events: none, behind content (z-index: 1).

Scroll Tracking

useScroll() from motion/react maps total scroll progress (0→1) to stroke-dashoffset (pathLength→0), progressively revealing the path.

Responsive Path

Path control points are calculated from actual section DOM positions via useRef + ResizeObserver. Curves recalculate on window resize.

Mobile

Path simplifies to gentler S-curves on smaller screens. Stroke-width reduces to ~6px. Hidden below 480px width for performance.
